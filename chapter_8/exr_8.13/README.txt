To adjust number to format 79XXXXXXXXX.

toDigit is function for deleting symbols which is not digits.

ndc is function to check that ndc(National Destination Code) starts from 9.

Examples:

1. If we have msisdn 7(999)XXX-XX-XX,
we will get msisdn 79XXXXXXXXX.

2. If we have msisdn 799943hj345,
than we will get result, that this is incorrect msisdn.
